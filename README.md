# VidSense AI

> AI-powered video editor — edit videos with natural language prompts or convert documents into narrated videos.

**Live Demo:** `https://vidsense-ai-85z2.vercel.app` &nbsp;|&nbsp; **Backend API:** `https://vidsense-ai-backend.onrender.com`

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔇 Silence Remover | Auto-detects and removes awkward pauses |
| 📝 Auto Captions | Whisper-powered subtitles burned into video |
| 👁️ Object Blur | YOLOv8 detects and blurs faces, cars, plates |
| 🛑 Word Censor | Replaces spoken words with beep tone |
| 🎵 Background Music | Emotion-aware music matched to video tone |
| 📑 Chapter Generator | Auto-generates YouTube-style timestamps |
| 📄 Doc → Video | Converts PDF/DOCX/TXT into narrated video |

---

## 🛠️ Tech Stack

**Backend:** FastAPI · LangGraph · Gemini 1.5 Flash · Whisper · YOLOv8 · MoviePy · FFmpeg · gTTS

**Frontend:** React · Vite · Tailwind CSS · Axios

**Deployed on:** Render (backend) · Vercel (frontend)
